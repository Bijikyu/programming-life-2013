    Class: Model.Cell

    Defined in: app/assets/javascripts/models/cell.js.coffee
    Inherits: Helper.Mixable
    Concerns: Mixin.DynamicProperties, Mixin.EventBindings, Mixin.TimeMachine

    Overview

    This is the model of a cell. It holds modules and substrates and is capable of simulating the modules for a timespan. A cell comes with one default module which is the Cell Growth.
